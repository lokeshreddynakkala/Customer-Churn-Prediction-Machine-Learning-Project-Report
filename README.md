# 📌 Project Overview

Customer churn refers to customers discontinuing a company’s service. In the banking sector, retaining customers is significantly more cost-effective than acquiring new ones.

This project develops a **Machine Learning model to predict whether a customer will leave the bank** based on demographic, financial, and behavioral attributes.

The project applies **data preprocessing, feature engineering, model training, and evaluation** to identify churn patterns.

---

# 🎯 Project Objectives

- Predict whether a customer will churn
- Identify key factors influencing customer attrition
- Build an accurate predictive model
- Provide insights that help improve customer retention strategies

---

# 📊 Dataset Information

The dataset contains **10,000 customer records** with the following features:

- Credit Score  
- Geography  
- Gender  
- Age  
- Tenure  
- Balance  
- Number of Products  
- Has Credit Card  
- Is Active Member  
- Estimated Salary  
- Exited (Target Variable)

### Target Variable

Exited  
- **0 → Customer stays**
- **1 → Customer churns**

# 🧠 Machine Learning Models Used

Several classification models were implemented and evaluated:

| Model | Accuracy |
|------|------|

| KNN | 84% |
| Support Vector Machine | 86% |
| Naive Bayes | 80% |
| Decision Tree | 84% |
| Random Forest | 85% |
| Gradient Boosting | **87%** |
| AdaBoost | 86% |

### 🏆 Best Performing Model

# 📈 Evaluation Metrics

Model performance was evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

These metrics ensure that the model effectively identifies customers who are likely to churn.

---

# 🔍 Key Insights

Important findings from the analysis:

- Customers aged **40–50** show higher churn probability
- **German customers** churn more frequently
- Customers with **multiple products** show increased churn risk
- Inactive customers have a higher chance of leaving
